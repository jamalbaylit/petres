## Overview

Petromod is a modern Python package for petroleum reservoir modeling and simulation. Built with performance and ease of use in mind, it provides tools for grid generation, fluid property calculations, and reservoir simulation workflows.

## Features

- **Grid Generation**: Create and manipulate structured and unstructured reservoir grids
- **Fluid Properties**: Calculate PVT properties for oil, gas, and water
- **Modern Python**: Built for Python 3.13+ with type hints and modern tooling
- **Extensible**: Modular design for easy customization and extension

## Installation

### Quick Install (from PyPI)

```bash
pip install petromod
```

### Install from Source

For development or latest features:

```bash
git clone https://github.com/jamalbaylit/petromod.git
cd petromod
uv pip install -e .
```

See the [installation guide](docs/source/getting_started/installation.rst) for detailed instructions.
