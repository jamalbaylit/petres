Grid Types
==========

Rectilinear Grid
----------------

A **rectilinear grid** is a structured grid where:

- Grid lines are straight and orthogonal
- Cells are rectangular or cuboidal

Spacing may vary along axes.

Regular Grid
------------

A **regular grid** is a special case of rectilinear grid where:

- Spacing is constant along each axis
- Δx, Δy, Δz may differ between axes

Cartesian Grid
--------------

A **Cartesian grid** is the most restrictive case:

- Equal spacing in all directions
- Cells are squares (2D) or cubes (3D)

Overview
--------

Hierarchy:

1. Rectilinear → flexible spacing
2. Regular → constant per axis
3. Cartesian → fully uniform
